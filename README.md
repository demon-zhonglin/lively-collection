## 🚀 How to Install / 如何安装

### English

1. **Prerequisite**: Ensure you have [Lively Wallpaper](https://github.com/rocksdanister/lively) installed.
2. **Download**: Clone this repository.
3. **Add to Lively**:
   - Drag and drop the downloaded file into the Lively Wallpaper window.
   - Or click **"Add Wallpaper"** and browse for the folder/file.

### 中文

1. **前提条件**：确保你已安装 [Lively Wallpaper](https://github.com/rocksdanister/lively)。
2. **下载**：克隆本仓库。
3. **添加到软件**：
   - 直接将下载的文件拖入 Lively Wallpaper 软件窗口中。
   - 或者点击软件内的 **“添加壁纸”**，浏览并选择对应的文件夹或文件。
